# Laravel SampleProject

## Requirements
PHP >= 8.1
Composer
MySQL або SQLite
Node.js & npm

## Installation

1. Clone project
2. Run commands:
```
make install
make seed-test-users 
```
// make seed-test-users - for test users

3. Run project
```
php artisan serve
```
